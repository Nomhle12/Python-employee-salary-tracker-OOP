Python-employee-salary-tracker-OOP

This project demonstrates an object-oriented Employee class using:
- Properties (`@property`)
- Setters with validation
- Encapsulation
- Class-level data structures
- Basic promotion and salary logic

Features
- Name validation
- Level validation (trainee → senior)
- Automatic salary assignment based on level
- Prevention of invalid level demotion
- Controlled salary updates
